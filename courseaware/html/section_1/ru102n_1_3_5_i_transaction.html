ITransaction

The ITransaction provides an interface for Redis Transactions. Transactions in Redis differ from transactions in other database, for a full description of transactions check out the transaction section of RU101.

The ITransaction interface is fundamentally an async interface. It exposes a very similar command set to the IDatabase, but it will only expose async versions of each command. That is because each command in ITransaction is async, as they will not be competed until after Execute is called.

You can get an instance of an ITransaction by calling IDatabase.GetTransaction() on your IDatabase object.
