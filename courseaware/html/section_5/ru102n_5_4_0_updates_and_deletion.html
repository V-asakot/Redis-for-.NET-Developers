Updating and Deleting Documents with Redis OM

In this section we'll look at performing the 'U' and 'D' of Updates, and Deletes, from the perspective of Redis OM. Unlike a traditional relational database redis-stack does not have the the ability to natively query, update, and delete records in isolation on the Redis Server, so we need to use the client to do so. Fortunately, Redis OM makes these tasks pretty straight forward as we'll be exploring in this section.
